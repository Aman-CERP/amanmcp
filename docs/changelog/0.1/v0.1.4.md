## Summary

F05 Metadata Store (SQLite) implementation completes Phase 1A. AmanMCP now has a complete foundation layer with project scaffolding, configuration, file scanning, gitignore parsing, and metadata storage.

---

## Added

- **F05: Metadata Store (SQLite)**
  - SQLite database with WAL mode for concurrent reads
  - Data models: Chunk, Symbol, File, Project
  - MetadataStore interface for storage abstraction
  - ON DELETE CASCADE for automatic cascading deletes
  - Prepared statements for efficient batch operations
  - Upsert support (INSERT OR UPDATE semantics)
  - Symbol search functionality
  - 13 comprehensive tests with 85.3% coverage
