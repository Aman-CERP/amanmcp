# Version 0.1.33

**Release Date:** 2026-01-02
**Phase:** Maintenance Mode - Resource Safety

## Summary

First maintenance release focused on resource safety. Prevents memory exhaustion, goroutine leaks, and security issues from symlinks.

---

## Fixed

### DEBT-003: Scanner Channel Abandonment
- Verified context-aware channel sends already implemented
- Added 6 edge-case tests for cancellation scenarios
- Files: `internal/scanner/scanner.go`, `internal/scanner/scanner_test.go`

### DEBT-001: Gitignore Cache Unbounded
- Verified LRU cache already implemented (hashicorp/golang-lru, 1000 entries)
- Added 2 tests verifying bounded cache behavior
- Files: `internal/scanner/scanner.go`, `internal/scanner/scanner_test.go`

### BUG-002: File Size Validation Missing
- Added `MaxFileSize` to `CoordinatorConfig` (default 100MB)
- Added size validation before `os.ReadFile()` to prevent memory exhaustion
- Removed duplicate `os.Stat()` call
- Added 2 tests
- Files: `internal/index/coordinator.go`, `internal/index/coordinator_test.go`

### BUG-005: Symlink Handling Missing
- Changed `os.Stat()` to `os.Lstat()` to detect symlinks
- Added symlink check with `os.ModeSymlink`
- Symlinks skipped gracefully with debug log
- Added 2 tests
- Files: `internal/index/coordinator.go`, `internal/index/coordinator_test.go`
