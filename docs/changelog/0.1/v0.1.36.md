## Fixed

### DEBT-013: Negative Cursor Accepted
- Added validation in `ListChunks()` to reject negative cursor offsets
- Returns descriptive error message for invalid cursor values

### DEBT-005: MarkdownChunker Missing Close
- Added `Close()` method to `MarkdownChunker` for interface consistency
- Currently no-op but ensures consistent Chunker interface

### DEBT-014: File Close Error Suppressed
- File close errors now logged instead of silently ignored
- Uses `slog.Warn()` for non-critical but notable failures

### DEBT-012: Nil vs Empty Slice Returns
- Standardized 10 functions across 7 files to return `[]T{}` instead of `nil`
- Improves API consistency and JSON serialization behavior
- Added 9 test cases to verify empty slice behavior
