## Summary

F08 Markdown Chunker implementation and validation. Completes the chunking subsystem with support for markdown documentation files alongside code files.

## Added

- F08: Markdown Chunker
  - Header-based section splitting (H1-H6)
  - Header path tracking (breadcrumb navigation)
  - YAML frontmatter extraction
  - Atomic block preservation (code blocks, tables, lists, MDX components)
  - Large section splitting at paragraph boundaries
  - Code fence metadata preservation
  - 19 tests with 88.2% coverage
