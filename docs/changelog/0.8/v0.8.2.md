# v0.8.2 - Data-Driven Configuration

## Added

- **FEAT-UNIX2**: RRF weights (`bm25_weight`, `semantic_weight`) are now configurable via `.amanmcp.yaml`
- **FEAT-UNIX2**: RRF constant (`rrf_constant`) is now configurable via `.amanmcp.yaml`
- **FEAT-UNIX2**: Environment variable `AMANMCP_RRF_CONSTANT` for overriding RRF k parameter

## Fixed

- Validation tests now skip gracefully when no index exists (fixes CI failures)

---

## Configuration

New search configuration options:

```yaml
search:
  bm25_weight: 0.65      # Default BM25 weight (0.0-1.0)
  semantic_weight: 0.35  # Default semantic weight (0.0-1.0)
  rrf_constant: 60       # RRF fusion constant (k parameter)
```

Environment variable overrides:
- `AMANMCP_BM25_WEIGHT` - Override BM25 weight
- `AMANMCP_SEMANTIC_WEIGHT` - Override semantic weight
- `AMANMCP_RRF_CONSTANT` - Override RRF constant

## Upgrade Notes

Existing configurations are automatically upgraded with new defaults via `MergeNewDefaults()`.
