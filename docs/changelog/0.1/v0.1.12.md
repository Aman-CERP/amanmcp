# Version 0.1.12

**Release Date:** 2025-12-29
**Phase:** Phase 2 - MCP Integration (17% complete)

---

## Summary

F13 Hybrid Search Engine validated - first feature of Phase 2 complete. Combines BM25 keyword search and vector semantic search with parallel execution, graceful degradation, and result fusion.

---

## Added

- F13: Hybrid Search Engine
  - SearchEngine interface (Search, Index, Delete, Stats, Close)
  - Parallel BM25 + vector search using errgroup
  - Graceful degradation (BM25-only, vector-only, or hybrid)
  - Result fusion with weighted scoring (BM25: 0.35, Semantic: 0.65)
  - Result enrichment from MetadataStore
  - Filter support (content type, language, symbol type)
  - Context timeout handling
