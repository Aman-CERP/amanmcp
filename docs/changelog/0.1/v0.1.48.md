## Technical Details

### BUG-043: Dimension Mismatch Fix

**Problem:** Auto-reindexing during `amanmcp serve` failed with "dimension mismatch: expected 4096, got 768" because serve/search commands used a hardcoded static embedder while index used config-based Ollama.

**Solution:** Changed serve.go and search.go to use config-based embedder selection (same as index command). Deprecated `NewDefaultEmbedder()` function with warning.

**Files Modified:**
- `cmd/amanmcp/cmd/serve.go` (2 locations)
- `cmd/amanmcp/cmd/search.go`
- `internal/embed/factory.go`

See [RCA-009](../postmortems/RCA-009-dimension-mismatch-autoreindex.md) for full analysis.
