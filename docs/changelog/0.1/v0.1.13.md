## Summary

Implemented F14 RRF Score Fusion - Reciprocal Rank Fusion algorithm for combining BM25 and vector search results with deterministic tie-breaking and score normalization.

---

## Added

- **F14: RRF Score Fusion** - Rank-based result fusion
  - Reciprocal Rank Fusion algorithm: score(d) = Σ weight_i / (k + rank_i)
  - Configurable k parameter (default: 60, industry standard)
  - Default weights: BM25 0.35, Semantic 0.65
  - Deterministic tie-breaking: RRFScore → InBothLists → BM25Score → ChunkID
  - Missing rank handling for documents appearing in only one list
  - Score normalization to 0-1 range
  - Original scores preserved in FusedResult struct
  - Integration with F13 Hybrid Search Engine
