## Validated

### F28: Monorepo Scope Filtering

Path-based filtering to limit search results to specific directories:

- **CLI `--scope` Flag**: Filter by path prefix (repeatable)
- **MCP Scope Parameter**: Array parameter for all search tools
- **Multiple Scopes OR Logic**: Match any of the specified scopes
- **Invalid Scope Handling**: Returns empty results, not error
- **Case-Sensitive Matching**: Proper directory boundary detection
