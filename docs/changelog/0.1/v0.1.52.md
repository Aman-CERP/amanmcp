- Fix GPU not stopping after Ctrl+C in `index` command (BUG-045)

- Fix ETA fluctuation during indexing progress
  - Add exponential smoothing (0.3 factor) to ETA calculation
  - ETA now changes smoothly instead of jumping wildly
