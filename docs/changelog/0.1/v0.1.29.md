### Configuration

```yaml
# .amanmcp.yaml
submodules:
  enabled: true          # Enable submodule discovery (default: false)
  recursive: true        # Discover nested submodules (default: true)
  include: []            # Only include these submodules (empty = all)
  exclude:               # Exclude these submodules
    - "vendor/*"
    - "third_party/*"
```
