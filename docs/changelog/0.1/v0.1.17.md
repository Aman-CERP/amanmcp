## Added

- F18: MCP Resources - expose indexed files to AI clients
  - Resource registration via MCP SDK's `AddResource()` method
  - `RegisterResources()` - loads indexed files and registers with MCP server
  - `handleReadResource()` - reads file content with security validation
  - MIME type detection: 30+ file extensions + special filenames
  - Path traversal prevention via `isValidPath()` validation
  - Index-only access (only files in metadata store accessible)
  - 1MB file size limit for resources
  - Cursor-based pagination in `ListFiles()` metadata method
  - 8 resource tests + 49 MIME type tests
