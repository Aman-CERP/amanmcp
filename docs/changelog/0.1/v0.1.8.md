## Added

- F09: Embedding Provider (Ollama) - local embeddings for semantic search
  - OllamaEmbedder with 768-dimensional vectors
  - Vector normalization for cosine similarity
  - Batch embedding with parallel processing
  - Model fallback (nomic-embed-text-v2-moe -> nomic-embed-text)
  - Retry with exponential backoff (up to 3 attempts)
  - Timeout handling (30s default)
  - Connection pooling with configurable pool size
  - Thread-safe with sync.RWMutex
  - 22 tests with 80.2% coverage

- Setup Command (`amanmcp setup`)
  - Interactive and non-interactive modes (`--check`)
  - OS-specific installation instructions (macOS, Linux, Windows)
  - Auto model download with progress bar
  - Primary/fallback model support
  - Signal handling (Ctrl+C)
  - 14 tests with 68.0% coverage
