# v0.3.3

**Release Date:** 2026-01-13

---

## Changed

- **MLX Default on Apple Silicon**: MLX is now the default embedding backend on Apple Silicon Macs (~1.7x faster). Ollama remains default on other platforms. Use `--backend=ollama` to override.

## Documentation

- Update Quick Start guide to show MLX as default for Apple Silicon
- Update CLI help text for `--backend` flag
- Update Makefile help to reflect platform-specific defaults

---

## Migration Notes

**Existing Ollama-built indexes**: May show dimension mismatch warnings when MLX becomes default. Run `amanmcp index --force .` to rebuild, or use `--backend=ollama` to continue with Ollama.
