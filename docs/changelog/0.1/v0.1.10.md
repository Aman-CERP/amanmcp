**Release Date:** 2025-12-28
**Phase:** Phase 1B - Core Search (86% complete)

## Summary

F11 BM25 Index implementation using Bleve v2. Provides keyword search with code-aware tokenization for hybrid search.

---

## Added

- F11: BM25 Index - keyword search for hybrid retrieval
  - BleveBM25Index using Bleve v2.5.7
  - Code-aware tokenizer (camelCase, snake_case splitting)
  - Stop word filtering (programming keywords)
  - Thread-safe with mutex protection
  - Persistence via Bleve disk storage
  - 16 BM25 tests + 8 tokenizer tests + 3 benchmarks
