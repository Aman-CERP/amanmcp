## Why These Changes?

### Problem: Documentation Search Pollution

BM25 keyword search cannot distinguish "documentation about code" from "actual code." When project management directories (like `.aman-pm/`) contain code examples, they can outrank actual implementations in search results.

### Solution: Sensible Defaults

The project configuration template now includes commonly-needed exclusion patterns by default:

```yaml
paths:
  exclude:
    - ".claude/**"    # Claude Code
    - ".cursor/**"    # Cursor IDE
    - ".aman-pm/**"   # AI-native PM
    - "archive/**"    # Historical content
```

Users starting new projects get best practices automatically.
